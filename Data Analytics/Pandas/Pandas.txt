Pandas Package:
    ** Series
    ** Dataframe 

Pandas is one of the most powerful and widely used open-source libraries in Python for data analysis and data manipulation. It provides fast, flexible, and expressive data structures designed to work with structured (tabular, multidimensional, potentially heterogeneous) and time-series data.

ðŸ”¹ Key Features of Pandas
Data Structures:

Series: One-dimensional labeled array.

DataFrame: Two-dimensional labeled data structure (like a table or spreadsheet).

Handling Missing Data: Easy functions like .isnull(), .fillna(), .dropna().

Data Alignment and Indexing: Automatic and explicit data alignment.

Input/Output: Read/write support for:

CSV, Excel, SQL, JSON, HTML, Parquet, etc.

Data Wrangling: Merging, reshaping, selecting, as well as cleaning.

Data Aggregation: Group by functionality (groupby) for summarizing data.

Time Series Analysis: Handling and manipulating datetime data.
